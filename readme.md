# @lucy/snowpack-plugin

Use [Lucy](https://lucylang.org/) files in a Snowpack project.

## Install

```shell
npm install @lucy/snowpack-plugin --save-dev
```

## Configuration

__snowpack.config.js__

```js
module.exports = {
  // ...
  plugins: [
    '@lucy/snowpack-plugin'
  ]
}
```
